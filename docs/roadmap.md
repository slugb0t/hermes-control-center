# Roadmap

## Phase 1 — Skeleton and observability shell

- Responsive Nuxt app shell
- PWA manifest/icon metadata
- Dashboard cards
- Live activity placeholder
- API health/status endpoints
- Security/architecture docs

## Phase 2 — Hermes read integration

- Session list and session detail viewer
- Gateway/log status
- Recent tool calls
- Cron job list
- KB root browsing in read-only mode

## Phase 3 — Controlled actions

- Approve/deny pending tool calls
- Send prompts from web UI
- Markdown editing for allowlisted KB folders
- Calendar read/create/edit with explicit delete confirmation
- Skills/memory/config viewers

## Phase 4 — Production polish

- Auth/session management
- Audit log
- SSE/WebSocket live streams
- Deployment docs
- Optional home-server deployment profile
